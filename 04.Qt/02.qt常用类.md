#### 8. QDataStream

> 处理二进制流 ：包括音频、图像等

以流的方式读写文本

```cpp
//写文件
QFile file("../test.txt");
if(file.open(QIODevice::WriteOnly))
{
    // 往数据流中写数据，相当于往文件里写数据
    QDataStream stream(&file);
    stream << QString("主要看气质") << 250;
    file.close();
}

//读文件
if(file.open(QIODevice::WriteOnly))
{
    QDataStream stream(&file);
    QString str;
    int a;
    stream >> str.totf8().date() >> a;//读文件的顺序和写文件的顺序一样
    qInfo() << str.toUtf8().data() << a;
    file.close();
}
```

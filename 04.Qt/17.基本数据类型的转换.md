### QString
#### 1、string、char*、const char*之间的转换
```cpp
// 1、string转const char*
string s="abc";
const char* c_s=s.c_str();

// 2、const char*转string
const char* c_s ="abc";
string s(c_s);

// 3、string转char*
string s ="abc";
char* c;
const int len=s.length();
c=newchar[len+1];
strcpy(c,s.c_str());

// 4、char*转string
char* c="abc";
string s(c);

// 5、const char*转char*
const char* cpc ="abc";
char* pc=new char[100];//足够长
strcpy(pc,cpc);

// qstring -> string
string s = qstr.toStdString();

// string -> qstring
QString qstr2 = QString::fromStdString(s)
```


## vocode 远程开发

> remote和本地的ssh都是打开的。

&emsp;&emsp;安装 remote ssh 和 remote Development，重启ide；  
&emsp;&emsp;SSH TARGET，点击设置，选择config文件位置；
&emsp;&emsp;点击左侧资源管理器，点击+，弹出的弹框中输入ssh wangbin@127.0.0.1，回车。  

&emsp;&emsp;实际我的~/.ssh/config文件内容如下：

```shell
Host ubuntu
    HostName 172.16.12.128
    User wangbin
    ForwardAgent yes
    Port 22
```

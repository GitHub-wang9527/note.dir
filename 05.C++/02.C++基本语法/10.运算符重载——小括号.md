# 运算符重载——小括号

标签（空格分隔）： C++

---

> * 在c/c++中。”()”操作符表示的是一个函数调用符号，同样，它只能够通过类的成员函数来重载

```cpp
#include <iostream>
class cls
{
public:
    cls()
    {
        printf("构造函数\n");
    }
    void operator() ()  //重载"()"操作符，"()"内无操作数
    {
        printf("HelloWorld!\n");
    }

    void operator() (const char* str) //重载"()"，"()"内的操作数是字符串
    {
        printf("%s", str);
    }
};

int main(void)
{
    cls cc;
    cc();
    cc("Hello Linux\n");
    //system("pause");
    return 0;
}
```

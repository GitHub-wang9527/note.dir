# C++设计模式——单例模式


在cpp文件中
```cpp
className* className::m_instance=NULL;
className* className::getInstance()
{
    if(m_instance==NULL)
    {
        m_instance=new className();
    }
    //使当前界面显示在最顶端
    m_instance->hide();
    m_instance->show();
    return m_instance;
}
//注意静态函数访问非静态成员变量
```
